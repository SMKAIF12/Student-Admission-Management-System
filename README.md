# Student Admission Management System Application Using MERN Stack
Front End: ReactJS,ReactQueries,LifeCycleHooks
Back End: NodeJS, ExpressJS, jwt, bcrypt
DataBase: MongoDB (Atlas)

# Features:
# Candidate Module:
- Candidate Can Apply for the Admission using their details and add list of preference of courses in colleges.
- Candidate can pay fee and confirm their application to be reviewed.
- Candidate can withdraw their application before the seat allocation is confirmed.
- Candidate can download the application as PDF.

# Admin Module:
- Admin can view and manage all the applications.
- Admin can allocate the seat in bulk.(allocation pool after the application window closes)
- Can Add Institutes and Courses.

- Pagination, Search, filter is done in client side.
- Used React Query for handling the requests to server.
